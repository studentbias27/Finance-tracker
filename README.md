# 💰 My Finance Tracker

A simple command-line finance tracker built using Python.

## 🚀 Features

* Track Income & Expenses
* Category-wise breakdown
* Summary using NumPy
* CSV-based storage using Pandas
* Overview of balance

## 🛠️ Tech Used

* Python
* Pandas
* NumPy

## ▶️ How to Run

```bash
python expense_tracker.py
```

## 📂 Files

* `my_income.csv`
* `my_expenses.csv`
